Configuration des Services

1. Serveur Web Apache

Pour héberger ce site localement, j'ai installé le serveur HTTP Apache. La configuration par défaut se trouve dans le dossier /var/www/html.

2. Environnement de Développement (IDE)

J'utilise PhpStorm pour coder. J'ai configuré le déploiement automatique pour que mes fichiers modifiés soient directement envoyés vers le serveur Apache.

3. Gestion des Utilisateurs

Conformément au sujet, j'ai créé les utilisateurs suivants :

  • Un administrateur : admin_Gr04_Binome00
  • Deux utilisateurs standards : Binome01_1 et Binome02_2
  • Un groupe d'utilisateurs dédié : Binome04 (qui contient les utilisateurs ci-dessus).

J'ai également personnalisé l'invite de commande (prompt) pour afficher le nom de l'utilisateur courant.
